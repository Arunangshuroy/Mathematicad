# Parametric Mechanical Parts

Reusable mechanical components designed for FDM and laser-cut fabrication.

## Files
| File | Purpose |
|------|---------|
| `gear-parametric.scad` | Involute spur gear — configurable module, teeth, pressure angle |
| `bracket-adjustable.scad` | Slotted L-bracket with parametric hole pattern |
| `hinge-livinghinge.scad` | Living hinge for sheet/panel assemblies — 0.6 mm flex zone |
