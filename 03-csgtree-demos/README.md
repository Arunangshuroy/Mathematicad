# CSG Tree Demonstrations

Focused examples of Constructive Solid Geometry operations in OpenSCAD,
each with detailed comments explaining why each operation was chosen.

## Files
| File | Demonstrates |
|------|-------------|
| `boolean-ops-showcase.scad` | union · difference · intersection — annotated side-by-side |
| `minkowski-rounding.scad` | Rounded boxes via minkowski() vs hull() — performance comparison |
| `hull-examples.scad` | Convex hull for organic transitions and cable guides |
