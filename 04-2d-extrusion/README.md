# 2D-to-3D Extrusion Workflows

Demonstrates linear_extrude and rotate_extrude — particularly useful
when the profile originates from a DXF file (AutoCAD/laser workflow).

## Files
| File | Demonstrates |
|------|-------------|
| `dxf-import-extrude.scad` | Import DXF profile → linear_extrude to 3D solid |
| `rotate-extrude-vase.scad` | Lathe-style body via rotate_extrude of 2D profile |
| `linear-extrude-profile.scad` | Structural extrusion with twist and scale parameters |
